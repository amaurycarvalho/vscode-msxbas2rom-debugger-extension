# US-003.3 - Stop Debugging

## Story

As an MSX-BASIC developer, I want to stop debugging so the emulator session terminates cleanly.

## Acceptance Criteria

- Stop shuts down the emulator control session.
- The debug session terminates without errors.

## BDD Scenarios

### Scenario 1: Stop while program is running

Given the program is running
When the user executes Stop
Then the emulator session must be stopped
And the debug session must terminate
And no error should occur

### Scenario 2: Stop while program is paused

Given the program is paused
When the user executes Stop
Then the emulator session must be stopped
And the debug session must terminate
And no error should occur

## Technical specification

Use the following guidelines:

- Stopping ends the openMSX session and emits termination to the client.

Procedure when stopping:

1. Send `quit` to openMSX control and stop the process.
2. Transition the debug session to terminated state.
3. Emit a terminated event to the client.
